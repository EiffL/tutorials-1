Data and text files to support the "Building Astrometry.net Index Files" and "FITS Binary Tables" tutorials:

cat_cdfile.txt:  	Column definitions for the SDSS Stripe 82 Standard Star Catalog
VarStars_cat.txt:	Small catalog of well-known variable stars
VarStars_colDef.txt:	Column definitions for variable star catalog

Data in the file VarStars_cat.txt adapted from the following sources, and references therein: 

O'Donoghue, D., Warner, B. & Cropper, M. 1992, MNRAS, 258, 415
McGraw, J.T. 1977, ApJ, 214, L123
Stover, R.J., et al. 1980, ApJ, 240, 865
